# GPT_Playground
One to explore what GPT can do with CodeX.

## 火影忍者：中忍考试篇（文字版跑团）
本仓库新增了一个遵循规则书的文字跑团体验，自动化 DM 可以在 Windows 或其他平台上运行，负责掷骰、检定与战斗结算，玩家只需做出选择即可体验故事。

### 环境要求
- Python 3.10+
- 无需额外依赖（仅使用标准库）

### 运行方式
交互式体验：
```bash
python -m game
```

快速演示（使用预设角色与脚本选择）：
```bash
python -m game --demo --seed 42
```

### 玩法概述
- **角色创建**：根据选择的职业与背景自动分配 2024 版 D&D 标准阵列能力值，套用木叶/砂隐/音忍背景加值，并计算生命值与查克拉。木叶角色在长休后会自动获得英雄灵感。
- **英雄灵感**：在关键检定后可选择消耗英雄灵感重掷，体验新版规则的后验重掷机制。
- **阶段流程**：
  1. **笔试**：进行知识与作弊检定，可通过鸣人式宣言获得英雄灵感。
  2. **死亡森林**：三天时间内根据行动触发随机事件表，自动处理卷轴获取、伤害与疲劳。
  3. **淘汰赛/决赛**：与两名对手对决，通过体术、速度与意志检定结算结果。
- **自动化 DM**：
  - 使用 `random` 模块执行 d20/d6 等掷骰，自动对比 DC 判定成功与失败。
  - 根据事件自动扣除生命值/查克拉、授予卷轴或英雄灵感，并在体力耗尽时终止游戏。

### 文件结构
- `game/character.py`：角色数据结构、背景/职业能力分配与恢复逻辑。
- `game/dice.py`：通用掷骰与检定工具。
- `game/events.py`：三大阶段的事件流程、随机遭遇与战斗结算。
- `game/main.py`：命令行入口，支持交互模式与 `--demo` 演示模式。
